Цель проекта: Создать сайт, позволяющий в реальном времени просматривать данные о звонках работников компании.

Функции программы:
    По одной ссылке ("/" или "/show_data") можно просмотреть данные о звонках работников компании.
    По второй ссылке ("/add_data") можно добавить данные о звонке: Номера A и B (кто звонил и кому позвонил), начало звонка, время звонка в секундах, тип звонка (входящий/исходящий).
    По третьей ссылке ("/show_more_data") можно посмотреть данные о звонках работников за определенный месяц.
    По четвёртой ссылке ("/change_data") можно изменять данные о пользователях: добавить работника, изменить фамилию/номер работника, удалить работника.

У каждого работника будет айди, которое создаётся на один номер и фамилию в базе данных.
В показе данных будет видно дату, все данные каждого пользователя за сегодня и желательно выделять тех, у кого слишком маленькая продуктивность. Обновляться данные будут каждые 5 минут. Данные будут показываться в таблице вида [Работник|Всего звонков/Входящие/Исходящие|Общая длительность]. Все данные будут сохранены в базу данных.


При создании проекта я буду использовать: язык программирования Python, PyCharm и библиотеки Flask. Когда проект будет закончен список библиотек дополнится.
